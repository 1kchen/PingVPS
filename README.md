# PingVPS
A ping script to test VPS latency (Linode, Vultr, AWS).  
## Instructions  
> git clone https://github.com/1kchen/PingVPS.git  
> cd PingVPS  
> chmod +x pingVPS.sh  
> ./pingVPS.sh  
An optional argument can be accepted as the ping frequency.  
For example:  
> ./pingVPS.sh 100  
All the results will be there in several mins.
